# Frequently Asked Questions

## How do I install React Fast Bento?

See the [Getting Started](./Getting%20Started.md) guide for installation instructions.

## Can I use custom components inside BentoGridItem?

Yes! You can pass any React node as children.

## How do I customize the grid layout?

Use the `columns`, `gap`, and `span` props to control the layout.

## The grid doesn't look right on mobile. What should I do?

Check your CSS for conflicting styles and ensure you are using responsive settings.

## Where can I get help?

Open an issue on GitHub or ask in the community channels listed in the docs.
