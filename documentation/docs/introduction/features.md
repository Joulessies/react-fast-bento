# Features

React Fast Bento offers a rich set of features to help you build beautiful, responsive Bento Grid layouts with ease.

## 🍱 Easy to Use

Simple API for creating complex Bento Grid layouts:

```jsx
<BentoGrid>
  <BentoGridItem title="Welcome" />
  <BentoGridItem title="Profile" />
</BentoGrid>
```

## ⚡️ Performance Optimized

Built with performance in mind, using efficient rendering and minimal re-renders.

## 💅 Beautiful by Default

Comes with attractive default styles so your layouts look great out of the box.

## 🎨 Customizable

Easily override styles and layout options to match your brand or design system.

## 📱 Responsive

Works seamlessly across all device sizes. Try resizing your browser!

## 🧩 Component-Based

Built with React components for easy integration and composition.
